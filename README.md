# 数字图像处理平台

## 项目概述

这是一个基于Python的数字图像处理平台，集成了多种传统图像处理算法。该平台提供了直观的图形用户界面，方便用户进行图像处理操作。

## 功能特性

### 1. 几何运算
- 图像平移
- 图像旋转
- 图像缩放
- 图像镜像
- 图像裁剪
- 透视变换

### 2. 对比度增强
- 线性灰度变换
- Gamma校正
- 直方图均衡化
- 自适应直方图均衡化（CLAHE）
- 对比度拉伸
- 直方图匹配
- Retinex图像增强

### 3. 平滑滤波
- 均值滤波
- 中值滤波
- 高斯滤波
- 双边滤波
- 频率域低通滤波
- 维纳滤波
- 各向异性扩散滤波
- 非局部均值滤波

### 4. 图像分割
- 边缘检测（Canny、Sobel、Prewitt、Laplacian、LoG）
- 阈值分割（全局、Otsu、自适应、多级Otsu）
- 区域生长分割
- 分水岭算法分割
- K-means聚类分割
- GrabCut算法分割
- 主动轮廓模型分割

### 5. 其他功能
- 图像直方图分析和显示
- 实时预览处理效果
- 操作历史记录（撤销/重做）
- 批量处理支持
- 多种图像格式支持

## 系统要求

### 软件要求
- Python 3.8+
- 以下Python库：
  - OpenCV
  - NumPy
  - Pillow
  - Matplotlib
  - SciPy
  - scikit-image
  - tkinter（通常随Python安装）

### 硬件要求
- 推荐：2GB以上内存
- 推荐：支持OpenGL的显卡
- 硬盘空间：100MB以上可用空间
